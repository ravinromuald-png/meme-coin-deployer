# MemeCoin Deployer

Application permettant de déployer des coins Meme sur Solana gratuitement,  
et d'accepter des paiements en frais pour les autres utilisateurs.

## Fonctionnalités

- Déploiement gratuit de tokens Meme (via smart contract)
- Paiement par utilisateur pour déployer un coin
- Interface web simple avec solana wallet integration

## Comment utiliser ?

1. Cloner ce dépôt : `git clone https://github.com/tonomme/meme-coin-deployer.git`
2. Installer les dépendances : `npm install` (ou `yarn`)
3. Configurer ton wallet Solana
4. Lancer l'application : `npm start`

## Déploiement

Le dépôt contient un contrat Solana et une interface web.
